## Terraform module to build the infra

# How to run ?

```bash
    # export the profile 
    export AWS_PROFILE = <profile-name>

    # Run the tf init
    terraform init

    # Run plan
    terraform plan

    # Run apply
    terraform apply

```

# How to tear down

```bash
    # export the profile 
    export AWS_PROFILE = <profile-name>

    # Run the tf init
    terraform init

    # Run the destroy
    terraform  destroy
```
